# Visualizations Overview

This folder contains exported charts generated during the analysis. Each visualization supports a specific analytical question and illustrates patterns in Instagram engagement behavior.

---

## Funnel Engagement Analysis
**File:** 
![Funnel Engagement](funnel_summary.png)

This visualization displays how engagement progresses across interaction stages from impressions → likes → deep engagement (saves + shares).It is used to identify where audience interaction declines and determine which stage experiences the greatest behavioral drop-off.

---

## Media Type Performance Analysis
**File:**

![Media Type Analysis](media_type_analysis.png)

This chart compares average engagement metrics across content formats (carousel, photo, reel, video). It evaluates which media formats generate stronger engagement efficiency and sustain deeper interactions relative to impressions.
