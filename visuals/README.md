# Visual Analysis Summary

This folder contains key visual outputs from the Instagram engagement analysis. Each visual addresses a distinct business question related to audience attention, engagement efficiency, and content performance.

## 01. Engagement Funnel
Visualizes average audience movement from impressions through deeper engagement actions (reach, likes, comments, saves, shares). This chart identifies where the largest engagement losses occur across the funnel.

## 02. Engagement Efficiency by Media Type
Compares engagement rates (likes, saves, shares relative to reach) across content formats to evaluate which media types convert exposure into meaningful interaction.

## 03. Engagement Drop-Off by Media Type
Analyzes where engagement declines between funnel stages for each media type, highlighting which content formats lose audience attention most quickly.

## 04. High vs. Low Performing Content
Compares engagement quality metrics between top- and bottom-performing posts to identify characteristics associated with stronger audience retention and growth.
